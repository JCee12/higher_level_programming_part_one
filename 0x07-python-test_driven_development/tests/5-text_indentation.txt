# 5-text_indentation.txt

--------------------------------
How to use 5-text_indentation.py
--------------------------------

This module defines a text_identation function ''text_indetation(text)''.

Usage
-----

Text is printed with two new lines after each of the characters '', '', ''?'',
and '':''

::

     >>> text_indentation = _import_('5-text_indentation.').text_indentation
     >>> text_indentation("Hello?")
     Hello?
     <BLANKLINE>
