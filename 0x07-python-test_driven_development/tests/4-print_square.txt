# 4-print_square.txt

---------------------------
How to use 4-print_square.py
----------------------------

This module defines a square-printing function ''print_square(size)''.

Usage
-----

Squares are printed using the ''#'' character. The parameter ''size''
represents the height and width of the square

::

    >>> print_square(4)
